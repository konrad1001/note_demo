# Cebes

Cebes is a markdown based notepad tool, enhanced with proactive agentic capabilities. 

Cebes was developed as part of a third year final project. 

This project contains Cebes, a cross platform Flutter app, built for Windows, MacOS, Android and iOS, as well as two flutter packages that were built and deployed as part of the work.

Github: https://github.com/konrad1001/note_demo
